# Snake Duel

![Snake Animation](snakeanimation.gif)

A modern multiplayer Snake game built with vanilla JavaScript and PeerJS. Challenge your friends in real-time duels or play solo mode to improve your skills. Customize your snake's appearance and choose from multiple control schemes including arrow keys, WASD, or mouse controls.

## Features

- Real-time multiplayer gameplay via PeerJS
- Solo mode with unlimited scoring
- Customizable snake colors and names
- Multiple control options (Arrow Keys, WASD, Mouse)
- Dark, minimalistic UI design
- Host or join game lobbies with unique game codes

## How to Play

1. Customize your profile with a name, color, and preferred controls
2. Choose to Host a game, Join with a code, or play Solo
3. Collect food to grow your snake and increase your score
4. Avoid hitting walls, yourself, or your opponent
5. First to 10 points wins (unlimited scoring in solo mode)

## Technologies

- Vanilla JavaScript
- PeerJS for peer-to-peer connections
- HTML5 Canvas for rendering
- CSS3 for styling

## Getting Started

Simply open `index.html` in a modern web browser. No build process or dependencies required.
